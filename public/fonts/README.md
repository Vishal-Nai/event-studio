# Local Cursor Font Files

The app and frame SVGs use the licensed Cursor Gothic font files in this folder directly.

Expected local paths:

- `Cursor Gothic/WEB/WOFF2/CursorGothic-Regular.woff2`
- `Cursor Gothic/WEB/WOFF2/CursorGothic-Italic.woff2`
- `Cursor Gothic/WEB/WOFF2/CursorGothic-Bold.woff2`
- `Cursor Gothic/WEB/WOFF2/CursorGothic-BoldItalic.woff2`

Both the app UI and generated frame SVGs reference these files through
`@font-face`.
